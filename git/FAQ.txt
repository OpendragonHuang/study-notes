总结:必须使用git init命令创建仓库，执行git add . 和git commit（提交成功后）,再使用git branch命令，才显示出本地分支。

git branch ：查看本地分支
git branch -a :查看本地及远程仓库的分支
git branch --all ：查看本地及远程仓库的分支

“因为git的分支必须指向一个commit，没有任何commit就没有任何分支

提交第一个commit后git自动创建master分支” -------廖雪峰
